# Security Policy

## Supported Versions

Security updates are provided for the latest stable release line.

| Version | Supported |
| --- | --- |
| 1.x | Yes |
| < 1.0.0 | No |

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Report privately by email:

- `jeheskielventiokysunloy@gmail.com`

Include:

- A clear description of the issue.
- Reproduction steps and impact.
- Any proof-of-concept or logs that help triage.

You will receive acknowledgment as soon as possible, followed by status updates through triage, fix, and release.
